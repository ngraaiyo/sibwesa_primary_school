Excel File Template Guidelines

Your Excel file (.xlsx format) should have the following exact column headers in the first row:

    {# Styled list #}
  • 1 Prem_Number
  • 2 First_Name
  • 3 Middle_Name (Optional)
  • 4 Last_Name
  • 5 Subject_Code
  • 6 Score (Integer between 0 and 100)

Example:

{# Styled code block #} 
Prem_Number,First_Name,Middle_Name,Last_Name,Subject_Code,Score
KABAGE/25/001,John,K,Doe,MATH001,85
KABAGE/2025/002,Jane,,Smith,ENG001,72
KABAGE2025/001,John,,Doe,ENG001,90

Rows with missing Prem Number or Subject Code will be skipped.
Invalid Prem Numbers or Subject Codes will result in errors.
Scores outside the 0-100 range will be marked as errors.
Existing marks for the same student, subject, and examination will be updated.
